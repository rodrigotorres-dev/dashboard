# Projeto feito em **Next.JS**, Apendendo ferramenta como **Shadcn**

1. instalar o [Next Js](https://nextjs.org/) ```npx create-next-app@latest```
2. Instalar [Shadcn](https://ui.shadcn.com/docs/installation) ```npx shadcn@latest init```
3. iniciar o Projeto com ```**npm run dev**```

## Projeto Falhou na Inicialização? 

1. Execute o comando ```**npm i**```
